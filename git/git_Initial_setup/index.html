    git 초기설정

    2024-09-18 최대 1 분 소요

    On This Page

    • 모든 설정값 확인
    • 모든 설정값과 해당 위치 확인
    • 사용자 이름 및 이메일 설정
    • 설정값 지우기
    • 편집기 설정
    • 기본 Branch 설정

    모든 설정값 확인

    git config --list

    특정값 확인

    git config user.name

    모든 설정값과 해당 위치 확인

    git config --list --show-origin

    사용자 이름 및 이메일 설정

    git config --global user.name "<사용자ID>"
git config --global user.email {이메일}

    * global 옵션을 전달할 경우 한번만 수행해도 된다.

    설정값 지우기

    git config --unset user.name
git config --unset user.email

git config --unset --global user.name
git config --unset --global user.email

    * global로 설정한 경우, global로 지워야 한다.

    편집기 설정

    git config global core.editor emacs

    git config --global core.editor "'C:/Program Files/Notepad++/notepad++.exe' -multiInst -notabbar -nosessin -noPlugin"

    * Windows 시스템에서 다른 텍스트 편집기를 사용하려면 실행파일의 전체 경로를 지정해야 함.

    기본 Branch 설정

    git config --global init.defaultBranch main
