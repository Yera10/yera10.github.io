<SQL 예제 1>

`CAR_RENTAL_COMPANY_CAR` 테이블의 전체 데이터 조회
```SQL
SELECT *
    FROM CAR_RENTAL_COMPANY_CAR 
```

<SQL 예제 2>

`CAR_RENTAL_COMPANY_CAR` 테이블에서 `CAR_TYPE`이 SUV 인 데이터 조회
```SQL
SELECT *
    FROM CAR_RENTAL_COMPANY_CAR 
    WHERE CAR_TYPE = "SUV"
```

<SQL 예제 3>

`CAR_RENTAL_COMPANY_CAR` 테이블에서 `CAR_TYPE`이 SUV 인 차들의 `DAILY_FEE`의 평균 출력
```SQL
SELECT AVG(daily_fee)
    FROM CAR_RENTAL_COMPANY_CAR 
    WHERE CAR_TYPE = "SUV"  
```

<SQL 예제 4>

`CAR_RENTAL_COMPANY_CAR` 테이블에서 `CAR_TYPE`이 SUV 인 차들의 `DAILY_FEE`의 평균을 소수 첫번째 자리에서 반올림해 'AVERAGE_FEE' 로 출력
```SQL
SELECT ROUND(AVG(daily_fee), 0) as AVERAGE_FEE 
    FROM CAR_RENTAL_COMPANY_CAR 
    WHERE CAR_TYPE = "SUV"
```

<SQL 예제 5>

`ONLINE_SALE` 테이블에서 `user_id`, `product_id` 별로 개수 조회
```SQL
SELECT COUNT(*)
    FROM ONLINE_SALE
    GROUP BY user_id, product_id;
```

<SQL 예제 6>

`ONLINE_SALE` 테이블에서 `user_id`, `product_id` 가 동일한 데이터가 2개 이상 있는 `user_id`, `product_id`와 개수를 출력
```SQL
SELECT user_id, product_id, COUNT(*) as CNT
    FROM ONLINE_SALE
    GROUP BY user_id, product_id
    HAVING CNT > 1;
```

<SQL 예제 7>

`ECOLI_DATA` 테이블에서 자식이 있는 ID 들의 ID와 자식의 수를 출력
```SQL
SELECT PARENT_ID as ID, COUNT(*)
    FROM ECOLI_DATA
    WHERE PARENT_ID IS NOT NULL
    GROUP BY PARENT_ID;
```

PRODUCT 테이블에서 판매 중인 상품 중 가장 높은 판매가를  컬럼명 MAX_PRICE로 출력
